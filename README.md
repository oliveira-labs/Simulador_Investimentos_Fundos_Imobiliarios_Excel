# Simulador_Investimentos_Fundos_Imobiliarios_Excel
Planilha em Excel (projeto DIO.me) para simular aportes mensais e projetar patrimônio e dividendos em diferentes prazos (2–30 anos). Inclui parâmetros de rendimento, cenários e sugestão de alocação por perfis (conservador/moderado/agressivo) e tipos de FII.
